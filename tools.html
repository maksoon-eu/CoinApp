FreePriceBot
Invite the bot
You can add this pricebot for free to your telegram group by adding @Poocoin_Pricebot to your group.
Set the token
Set the tracked token with the command /set_token {address} For example, to set the poocoin token type: /set_token 0xb27adaffb9fea1801459a1a81b17218288c097cc
Request the price and chart
Anyone in the group can request the price chart by typing /price
Set additional text
You can set additional text at the end of each price response with /set_message {text}
You can set links inside the message with the HTML a tag. For example, in the preview below it has been used to set the social media links.
Banner Ads
You can place an anonymous advertisement for your project.

purchasing of

Go to the website
The banner ads are purchased through an ad provider a-ads.com
You can also purchase them by hovering on the icon in the top-right and clicking the "your ad here" link.
If your project is an un-released fair launch, or a presale that has no available presale url on your website, the ad will be limited to only appearing in the bottom left spot marked as "presale ads". It will appear in a-ads as being marked as "risky".
This limitation is removed after your token is launched or after you provide a presale url for unicrypt or dxsale.
